# Tradutor Morse

Desenvolvido para a cadeira de Estruturas de Dados, do 2º semestre de Ciência da Computação da UFRGS.

### Compilar

Para compilar, execute o seguinte comando:

```
$ gcc src/main.c src/abp.c src/avl.c -o morse.exe
```
